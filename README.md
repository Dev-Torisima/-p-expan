# MinusP（-p）拡張サポート  

-pは十分な学習なしで日本語でもプログラミングができる言語です  
この拡張は以下をサポートします  
・シンタックスハイライト（色付け）  
・スニペット  
・キーワードの補完／定数や関数名の補完  
  
  
# MinusP(-p) extended support  

-p language is programming language without learning well.  
This supports the following.  
- Syntax highlighting  
- Snippets  
- Keyword completion  
- Built‑in constants and functions
